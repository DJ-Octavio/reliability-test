机械产品可靠性设计与分析

本节聚焦机械产品的可靠性度量参数、概率设计法以及工程设计实践。核心思路是：先明确可靠性要求，再用失效机理和概率方法把“应力—强度”关系落实到设计中，最后通过评审、分析与增长验证形成闭环。

3.7.4

机械产品的可靠性度量参数

教材指出，机械产品一般可分为整机（系统）和零部件。适合机械产品可靠性的常用度量参数，无论是机械系统还是零部件，最常用的是：可靠度、寿命或可靠寿命、平均故障间隔时间/里程（MTBF）。

可靠度

指产品在规定条件下、在规定时间内不发生故障的概率。教材建议按照失效影响的重要性将可靠度分成 5 级，关键零件的可靠度许用值应取高值。

寿命 / 可靠寿命

机械产品都存在使用寿命问题。对设计寿命期内并非所有产品都能可靠工作的情形，可用“可靠度为给定值 R 时的工作寿命”来表征，轴承常以 R=0.9 的寿命作为参数。

MTBF / 里程

平均故障间隔时间/里程常用于机械整机或系统可靠性度量，其定义是工作总时间/里程与故障总次数之比。它能反映统计意义上的可靠性水平。

设计提示

机械系统的失效率通常非恒定，零部件寿命数据又较缺乏，因此 MTBF 往往需要结合大量统计或试验数据才能确定。

重点：机械零部件的可靠性度量不是“越多越好”，而是要根据故障后果、设计寿命和关键程度来选参数，尤其关键件要选更高的可靠度许用值。
3.7.5

概率设计法：机械可靠性定量设计的主方法

教材明确写到：概率设计法是机械产品可靠性定量设计的主要方法。它以应力—强度干涉模型和功能失效极限状态函数理论为基础，把载荷、强度、尺寸、寿命等看作服从一定分布的随机变量，再用概率方法计算失效概率和可靠度。

应力可能引起失效的一切因素组合，如载荷、温度、振动等
强度抵抗破坏或失效的一切因素组合，如材料、尺寸、结构等
可靠状态强度 > 应力
失效状态强度 < 应力

设计逻辑

  1. 先确定设计工况和可靠性要求。
  2. 再识别与设计相关的随机变量及其分布。
  3. 计算失效概率与可靠度，评估是否满足要求。
  4. 若不满足，则回到结构、尺寸、材料、工艺等环节调整设计。
工程含义：机械设计不能只做“名义值设计”，还要把随机波动纳入分析，否则很容易在实际使用中因应力超出强度而失效。
3.7.6

工程设计实践：把可靠性落到图纸与试验

机械产品可靠性设计分析的流程是教材强调的闭环：明确要求 → 调查相似产品 → 可靠性分配 → FMEA/FTA → 关键件定性/定量设计 → 计算分析 → 设计评审 → 可靠性增长。

1. 明确可靠性要求

要先明确定性和定量要求，如可靠度、寿命、平均故障间隔里程等，并结合寿命剖面、任务剖面和故障判别准则。

2. 分析相似产品

收集常见故障模式、故障频率、故障原因、成功经验和失败教训，提炼出可靠性设计准则。

3. 关键件定量设计

关键件、重要件除了借鉴成功经验，还应创造条件进行概率设计；先给定工况，再计算可靠度并迭代优化。

4. 可靠性增长

图纸转化为样机后，要通过试验暴露缺陷，再改进、验证，持续提升可靠性。

工程应用要点

  • 机械结构应尽可能简单，减少不必要环节与传力路线。
  • 关键设计变量要做灵敏度分析，关注外部条件变化。
  • 应进行应力—强度优化设计，找出最佳匹配。
  • 需考虑公差配合、表面粗糙度、环境防护、刚度与疲劳等因素。
  • 防止某个构件失效引起连锁失效，可采用止裂、多路传力、多重元件设计等。
记住一句话：机械产品可靠性不是“检验出来”的，而是从设计开始，通过分析、评审和增长试验“做出来”的。
